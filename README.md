# Estrutura de dados
# EDI-List4
Este repositório contém a Atividade 4 de exercícios da disciplina de Estrutura de Dados, desenvolvida em Java.
O projeto dá continuidade ao estudo da linguagem Java com foco em modularização (funções/métodos), tipos abstratos de dados (TAD) e análise de algoritmos.

---

## 📚 Exercícios

Esta lista tem como objetivos principais:

- Reescrever os algoritmos anteriores usando métodos bem definidos.
- Evitar o uso de variáveis globais, trabalhando com parâmetros e valores de retorno.
- Organizar o código para que o método main apenas orquestre as chamadas das funcionalidades.
- Implementar uma função para inverter vetores in-place (sem vetor auxiliar).
- Discutir a complexidade de tempo do algoritmo em notação O.

### Exercícios implementados

- **Funcao.java**  
- Neste arquivo esta todas as funçoes utilizadas na **Parte1.java** referente a atividades anteriores.

- **q2.java**  
- Neste arquivo é o main onde as funções crias em **Funcao.java** serão chamadas nesse main.

- **q3.java**  
- Neste programa temos um vetor ja definido ([10, 20, 30, 40, 50]) que por meio de uma função criada fora do main, sera invetido (inverterLinha).

- **Respostas Manuscrita**
- Neste arquivo esta todas a respostas manuscritas referente as questões manuscritas da atividade 4.
---

## 🚀 Recursos Implementados

O projeto inclui:

- Código em Java puro (sem frameworks).
- Organização por exercício, cada um em sua própria classe (Q1, Q2, ... ou nomes equivalentes).
- Uso de:
    - Métodos para separar responsabilidades (ex.: leitura, cálculo, exibição).
    - Passagem de parâmetros e retorno de valores, evitando variáveis globais.
    - Vetores (arrays) na manipulação de dados.
    - Uma função específica para inversão de vetor in-place.
